### 🚀 About Me

Every line of code is an attempt to bring order to chaos, yet the deeper I go, the more beautiful the chaos becomes. I create by dismantling, learn by failing, and grow by questioning what I once believed to be true. In the end, software isn't what I'm building—it's merely the evidence of a mind that refuses to stop exploring.

🌱 &nbsp;I'm currently learning **Security, Backend and Japanese**
